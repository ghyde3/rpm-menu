**SectionHeader** — big Anton caps board header for a menu category.

```jsx
<SectionHeader>Sandwiches</SectionHeader>
<SectionHeader color="var(--accent-secondary)" stars>Baskets · Dogs</SectionHeader>
<SectionHeader size="xl" align="center">Desserts</SectionHeader>
```

Red by default; flame orange for secondary sections. `stars` adds amber ★ flankers. Sizes `md | lg | xl`.
